# 🚀 Ikjas Rasool P — Developer Portfolio


## ✨ Features

- 🌌 **Live Starfield Canvas** — animated background with parallax stars, nebulas, and shooting stars
- ⚡ **Glitch Text Effects** — cyberpunk-style RGB glitch on headings
- ⌨️ **Typewriter Animation** — cycling role titles with smooth type/delete loop
- 🪐 **Orbital Animation** — rotating planet rings on the hero section
- 📜 **Scroll Reveal** — IntersectionObserver-based fade-up animations
- 🔍 **Project Filter** — filter 12+ projects by tech stack
- 📱 **Fully Responsive** — optimized for mobile, tablet, and desktop
- 🎨 **Custom Scrollbar & Selection** — cohesive deep-space aesthetic throughout

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/ikjasrasool/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
